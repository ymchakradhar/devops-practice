Stage 1 — GitHub Setup
✅ Complete
1
Created project folder
Created a dedicated folder for DevOps practice on D drive
mkdir "D:\My Personal\devops-practice" cd "D:\My Personal\devops-practice"
2
Configured Git identity
Set global username and email for Git commits
git config --global user.name "ymchakradhar" git config --global user.email "ym.chakradhar@gmail.com"
3
Initialized Git repository
Turned the folder into a Git repository
git init # Output: Initialized empty Git repository in D:/My Personal/devops-practice/.git/
4
Connected to GitHub
Linked local folder to remote GitHub repository
git remote add origin https://github.com/ymchakradhar/devops-practice.git
5
First commit and push
Added index.html, committed and pushed to GitHub
git add . git commit -m "my first devops commit" git push -u origin master
💡
Key Git Commands to Remember:
git add . → Stage all files  |  git commit -m "" → Save snapshot  |  git push → Upload to GitHub  |  git status → Check current state

Stage 2 — Docker
✅ Complete
1
Created Dockerfile
Defined how to package the app into a container using nginx
FROM nginx:alpine COPY index.html /usr/share/nginx/html/index.html
2
Built Docker image
Packaged the app into a Docker image
docker build -t devops-practice . # Successfully tagged devops-practice:latest
3
Ran Docker container
Started the container and mapped it to localhost:8080
docker run -d -p 8080:80 --name myapp devops-practice # App available at: http://localhost:8080
CommandWhat it does
docker psSee running containers
docker stop myappStop your container
docker start myappStart it again
docker rm myappDelete container
docker imagesSee all your images

Stage 3 — CI/CD Pipeline (GitHub Actions)
✅ Complete
1
Created workflow folder
mkdir .github mkdir .github\workflows
2
Created deploy.yml — GitHub Actions pipeline file
Location: .github/workflows/deploy.yml
name: CI/CD Pipeline on: push: branches: - master env: FORCE_JAVASCRIPT_ACTIONS_TO_NODE24: true jobs: build-and-deploy: runs-on: ubuntu-latest steps: - name: Checkout code uses: actions/checkout@v4 - name: Build Docker image run: docker build -t devops-practice . - name: Copy files to EC2 uses: appleboy/scp-action@v0.1.7 with: host: ${{ secrets.EC2_HOST }} username: ubuntu key: ${{ secrets.EC2_SSH_KEY }} source: "." target: "/home/ubuntu/devops-practice" - name: Deploy on EC2 uses: appleboy/ssh-action@v1.0.3 with: host: ${{ secrets.EC2_HOST }} username: ubuntu key: ${{ secrets.EC2_SSH_KEY }} script: | cd /home/ubuntu/devops-practice docker stop myapp || true docker rm myapp || true docker build -t devops-practice . docker run -d -p 8080:80 --name myapp devops-practice
